在解决这个任务之前，您可以尝试解决"Brackets" 任务。

您的任务是删除不必要的括号，恢复开放式和封闭式括号的平衡，同时尽量减少删除次数。

在给定的字符串中，只能使用 3 种括号"()", "[]", "{}" 。

只有括号可以关闭括号。也就是说，在这个表达式"(}" 中，括号是不平衡的。在空字符串（即不包含任何括号的字符串）中，括号是平衡的，但删除所有括号并不是最佳解决方案。

如果有多个正确答案，那么您应该选择可以删除的字符更靠近开头的那个。例如，在本例中"[(])" ，正确答案将是"()" ，因为可去掉的括号更靠近行首。

输出： 字符串字符串 (str{% if is_js %}ing{% endif %})字符串"(, ), {, }, [, ]"。
