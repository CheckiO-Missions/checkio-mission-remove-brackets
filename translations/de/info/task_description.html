Bevor Sie diese Mission lösen, können Sie versuchen, die Mission "Brackets" zu lösen.

Ihre Aufgabe ist es, das Gleichgewicht zwischen offenen und geschlossenen Klammern wiederherzustellen, indem Sie die überflüssigen Klammern entfernen und dabei versuchen, so wenig wie möglich zu löschen.

In der gegebenen Zeichenfolge können nur 3 Arten von Klammern "()", "[]", "{}" verwendet werden.

Nur eine Klammer kann eine Klammer schließen. Das heißt, in diesem Ausdruck "(}" - sind die Klammern nicht ausgeglichen. In einer leeren Zeichenkette, d.h. in einer Zeichenkette, die keine Klammern enthält, sind die Klammern ausgeglichen, aber das Entfernen aller Klammern wird nicht als optimale Lösung angesehen.

Wenn es mehr als eine richtige Antwort gibt, sollten Sie diejenige wählen, bei der das Zeichen, das entfernt werden kann, näher am Anfang liegt. In diesem Fall "[(])" lautet die richtige Antwort zum Beispiel "()", da die entfernbaren Klammern näher am Anfang der Zeile liegen.
