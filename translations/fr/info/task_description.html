Avant de résoudre cette mission, vous pouvez essayer de résoudre la mission "Brackets".

Votre tâche consiste à rétablir l'équilibre entre les parenthèses ouvertes et fermées en supprimant les parenthèses inutiles, tout en essayant d'utiliser le nombre minimum de suppressions.

Seuls 3 types de parenthèses "()", "[]", "{}" peuvent être utilisés dans la chaîne donnée.

Seule une parenthèse peut fermer une parenthèse. C'est-à-dire que dans cette expression "(}" - les parenthèses ne sont pas équilibrées. Dans une chaîne vide, c'est-à-dire dans une chaîne qui ne contient aucune parenthèse, les parenthèses sont équilibrées, mais la suppression de toutes les parenthèses n'est pas considérée comme une solution optimale.

S'il y a plusieurs réponses correctes, vous devez choisir celle où le caractère qui peut être supprimé est le plus proche du début. Par exemple, dans le cas présent, "[(])", la réponse correcte sera "()", car les parenthèses amovibles sont plus proches du début de la ligne.
